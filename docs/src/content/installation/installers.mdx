# Blockfrost platform official installers

<Tabs items={['Linux','macOS','Windows']}>
  <Tabs.Tab>

    For Linux, paste the following into your shell prompt:

    ```shell
    curl -fsSL \
         https://github.com/blockfrost/blockfrost-platform/releases/latest/download/curl-bash-install.sh \
         | bash
    ```

    This script will download and install the latest version of the Blockfrost platform.

    ## Manual installation

    Alternatively, you can download pre-built binaries directly from the [GitHub releases page](https://github.com/blockfrost/blockfrost-platform/releases/latest).

    After downloading, make the binary executable with `chmod +x blockfrost-platform` and move it to your preferred location.

  </Tabs.Tab>
  <Tabs.Tab>
  
    For macOS, paste the following into your macOS terminal:

    ```shell
    curl -fsSL \
         https://github.com/blockfrost/blockfrost-platform/releases/latest/download/curl-bash-install.sh \
         | bash
    ```

    ## Installing using Apple disk image

    Download the DMG image from the [GitHub releases page](https://github.com/blockfrost/blockfrost-platform/releases/latest) and install it by moving the image to your applications.

    ![DMG](/dmg.png)

    ## Installing using Homebrew

    If you are using [Homebrew](https://brew.sh/) on your macOS (or Linux), you just need to run:

    ```shell
    brew install blockfrost/tap/blockfrost-platform
    ```

  </Tabs.Tab>
  <Tabs.Tab>

    Download the Windows installer (the `.exe` file) from the [GitHub releases page](https://github.com/blockfrost/blockfrost-platform/releases/latest).

    ## Using the installer

    After downloading the `.exe` file, double-click it and follow the on-screen instructions to complete the installation.
    The installer will guide you through the setup process.

    ## Manual installation

    If you prefer manual installation:

    1. Download the `.zip` file from the GitHub releases page.
    2. Extract the contents to your preferred location.
    3. The extracted folder contains the `.exe` file.
    4. Double-click it and follow the on-screen instructions.

  </Tabs.Tab>
